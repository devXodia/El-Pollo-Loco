Class: ThrowableObject

ThrowableObject(x, y)

new ThrowableObject(x, y)

This constructor loades the Parent Class and loads the first Image. After that it sets the throwable bottle x and y position to the ones of the character. Then it sets the height of the Bottle. The Direction of the Bottle throw is being set to the characters currently looked at direction. And after that it calls the Throw function
Parameters:
Name Type Description
x number x position of character
y number y position of character
